【JDK, JRE, JVM区别】
JDK是Java编译，运行和类库工具包，Java源码的集成环境，是Java开发工程师必须安装的。
JRE是Java运行环境，运行字节码文件，如果只是运行Java程序，可以只安装JRE即可。
JVM是Java虚拟机，实际就是JRE的bin，JRE的lib目录是jvm运行时所依赖的类库。
