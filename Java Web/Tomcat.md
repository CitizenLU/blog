Tomcat主要目录的作用







| 目录    | 作用                                                   |
| ------- | ------------------------------------------------------ |
| bin     | 所有的可执行命令，启动和关闭服务器的命令就在此文件夹中 |
| conf    | 服务器的配置文件夹，保存了各个配置信息                 |
| lib     | Tomcat服务器所需要的各个库文件                         |
| logs    | 保存服务器的系统日志                                   |
| webapps | Web应用程序存放的目录，Web项目保存到此目录中即可发布   |
| work    | 临时文件夹，生成所有的临时文件（`*.java`、`*.class`）  |
